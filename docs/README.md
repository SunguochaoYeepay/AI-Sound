## 贡献指南

如需贡献文档，请遵循以下规则：

1. 将文档放在对应的子目录中
2. 使用 Markdown 格式编写文档
3. 为文档添加清晰的标题和目录
4. 提交前检查文档格式和链接
5. 更新本 README 文件，添加新文档的链接

## 文档标准

- 文件名使用小写字母和下划线
- 文档中的链接使用相对路径
- 代码示例应该有明确的语言标注
- 图表应当存放在各文档目录下的 `images` 子目录中
